Tags de Título - h1, h2, h3, h4, h5, h6

(Voltar)

No caso das tags de titulo, pode-se entender que existe uma escala de 1 a 6 (h1 a h6), onde a h1 predomina sobre as demais por representar o título principal, afinal, todo texto tem ao menos 1 título. As demais tags são utilizadas conforme os subtítulos inseridos, como no exemplo do sumário automático do Word, sendo assim, a h1 deve ser vista como a mais importante e a h6 a menos importante.

Tag de Paragrafo - tag P

(Voltar)

O parágrafo é o elemento base no texto editado. A tag para um parágrafo é P . Ele coloca uma linha vazia acima e uma abaixo do texto para fazè-lo evidente, e o navegador o terá pelo que ele é.

Tag de Destaque - tag Mark

(Voltar)

O elemento HTML mark define o texto que deve ser marcado ou destacado.

Tag de Formatação - Itálico, Negrito e Sublinhado

(Voltar)

Tag <.i><./i> Formata textos em itálico a tag <.n><./n> em negrito e a tag <.u><./u> em sublinhado. Ambas requerem abertura e fechamento com o conteúdo entre elas.

Tags de Criação de Listas - UL, OL e DL

(Voltar)

HTML tem TRÊS tipos de listas. <.ol> mostra as listas ordenadas, enquanto <.ul> , as desordenadas. E para fazer uma lista de definições use <.dl>.

Tags de Link

(Voltar)

  1. Href

    2. O atributo "href" nomeia a conexão com outra página de web. Na verdade é o lugar para onde serão enviados os usuários que clicarem no link.

  2. a

    3. Para marcar o início e o fim de uma âncora pode ser usado <.a><./a>. Escolha o tipo de atributo que você precisa e coloque dentro da tag.
